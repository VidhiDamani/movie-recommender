# 🎬 Movie Recommender System  

> *“Because sometimes… picking the next movie is harder than watching one.”* 🍿

> Link: https://movie-recommender-cos.streamlit.app/

Welcome to the **Movie Recommender System** — a simple, elegant, and interactive app built using **Streamlit** and **Python** that helps you discover movies similar to your favorites.  

---

## 🌸 Features  

✨ **Content-Based Recommendations** — Suggests similar movies using text similarity on overviews.  
🎀 **Pink & Black Aesthetic UI** — Clean, dark-themed interface with a pop of pink.  
🖱️ **Hover Effects** — See detailed movie info (overview, genre, etc.) on hover.  
🎞️ **Dropdown Selection** — Choose a movie from a smart dropdown list.  
⚡ **Fast & Lightweight** — Built using Pandas, Scikit-learn, and Streamlit.  

---

## 🧠 How It Works  

1. The dataset (`movies.csv`) contains movie titles and descriptions.  
2. Using **TF-IDF Vectorization**, the text data is converted into numerical form.  
3. **Cosine Similarity** measures how close one movie is to another based on their overview.  
4. The top similar movies are displayed instantly in the app.
